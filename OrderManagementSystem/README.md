# Система управления заказами на Spring Boot

## Описание проекта
Система автоматизации процессов приёма, обработки и оплаты заказов для интернет-магазина, разработанная на Spring Boot.

## Функциональные возможности
- Управление покупателями (CRUD операции)
- Управление товарами и складскими остатками
- Создание и обработка заказов
- Резервирование товаров
- Обработка платежей
- REST API с документацией Swagger
- Валидация данных
- Обработка исключений

## Технологический стек
- **Фреймворк:** Spring Boot 3.1.0
- **Язык программирования:** Java 17
- **База данных:** PostgreSQL
- **ORM:** Spring Data JPA (Hibernate)
- **Документация API:** Springdoc OpenAPI 3 (Swagger)
- **Сборка:** Maven
- **Валидация:** Jakarta Validation
- **Утилиты:** Lombok
